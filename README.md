# SingulEYE

Chrome extension for the `thesingularity` gallery list and view pages.

## Features

- Loads the current tab list from the page automatically.
- Lets you turn blocking ON or OFF for each head.
- Shows how many rows are currently hidden on the page.
- Lets you switch all heads ON, OFF, or back to the default set.
- Lets you turn floating-IP post visibility ON or OFF.
- Lets you turn floating-IP comment visibility ON or OFF inside post pages.
- Applies changes immediately without reloading the page.

## Scope

- `https://gall.dcinside.com/mgallery/board/lists/?id=thesingularity`
- `https://gall.dcinside.com/mgallery/board/lists/?id=thesingularity&exception_mode=recommend`
- `https://gall.dcinside.com/mgallery/board/view/?id=thesingularity&no=*`

## Install

1. Open `chrome://extensions`.
2. Turn on `Developer mode`.
3. Click `Load unpacked`.
4. Select the extension folder in this workspace.
